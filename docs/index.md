# Bash2Yaml

Bash2Yaml aims to solve the pain of editing bash when it is in a YAML string. It supports multiple CI/CD platforms
including **GitLab CI** and **GitHub Actions**, with more targets planned.

Get conceptual background looking at the "Who is this for?"

Get started by looking at the compile and decompile commands, and check out
[Supported Targets](targets/supported_targets.md) to see which CI/CD platforms are available.
